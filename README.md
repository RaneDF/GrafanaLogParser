# Grafana Log Beautifier

This Tampermonkey userscript beautifies VictoriaLogs raw JSON output inside Grafana Explore:

- Pretty formatted message + stack trace
- Metadata header
- Collapsible stack trace
- Hide level (ERROR/WARN/INFO/DEBUG)
- Remove *all* Grafana placeholder rows
- Sticky toolbar
- Works in exported HTML files too
